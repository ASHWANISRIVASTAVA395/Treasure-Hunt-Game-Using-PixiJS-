# Treasure Hunt Game Using PixiJS

## Overview

A browser-based Treasure Hunt Game developed using PixiJS and HTML5 Canvas.

The player must collect all coins, obtain the key, avoid the enemy, and reach the treasure chest before the timer expires.

## Features

* Start Screen
* Player Movement using Arrow Keys
* Coin Collection System
* Key Collection System
* Enemy Obstacle
* Countdown Timer
* Treasure Chest Objective
* Win/Lose Conditions
* HTML5 Canvas Rendering
* PixiJS Framework

## Technologies Used

* PixiJS
* JavaScript
* HTML5 Canvas
* CSS

## Controls

* Arrow Keys → Move Player
* Enter → Start Game

## Game Objective

1. Collect all 5 coins.
2. Collect the key.
3. Avoid the enemy.
4. Reach the treasure chest.
5. Complete before the timer ends.
